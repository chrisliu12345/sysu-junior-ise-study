\begin{frame}
	\frametitle{什么是兼容}
	\begin{itemize}
		\item 二进制兼容：对于一个可执行文件，不需要做任何修改就可以直接运行
		\item 源码兼容：在新的系统上，不需要更改源码，只需要重新编译或利用解释器即可运行
		\item 向前兼容：新程序能接受老程序的输入
		\item 向后兼容：老程序能接受新程序的输入
	\end{itemize}
\end{frame}
